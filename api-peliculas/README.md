Preparation
---
```console
$ python3 -m venv venv
$ . venv/bin/activate
(venv) $ pip install -r requeriments.txt
```
Execution
---
```console
$ . configuration.sh 
$ flask db init
$ flask db migrate -m "Initial DB"
$ flask db upgrade
$ flask run
```
